# Jewellery Generator

A modern web application for generating jewellery designs.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## How to Use

1. Navigate to the main page of the application.
2. Use the provided tools and options to customize and generate your jewellery designs.
3. The interface will guide you through selecting materials, styles, and other properties.
